# Keyboard Shortcuts

## Navigation

| Keys                    | Action                                                        |
| ----------------------- | ------------------------------------------------------------- |
| `Left/Right`            | Move between data, operations, and buttons within a statement |
| `Up/Down`               | Move between statements                                       |
| `Alt + Left/Right`      | Jump between data and operations, skipping text               |
| `Alt + Up/Down`         | Jump to parent context                                        |
| `Cmd/Ctrl + Left/Right` | Jump to first/last element in current statement               |
| `Cmd/Ctrl + Up/Down`    | Jump to first/last statement of the operation                 |

## Editing

| Keys                                    | Action                          |
| --------------------------------------- | ------------------------------- |
| `Cmd/Ctrl + Z`                          | Undo                            |
| `Cmd/Ctrl + Shift + Z` / `Cmd/Ctrl + Y` | Redo                            |
| `Alt + =`                               | Add operation call              |
| `Ctrl + Space`                          | Open dropdown                   |
| `Backspace` / `Alt + Backspace`         | Delete focused element          |
| `Enter`                                 | Toggle boolean / confirm input  |
| `Escape`                                | Close dropdown or details panel |

## Sidebar

| Keys           | Action         |
| -------------- | -------------- |
| `Ctrl+Shift+1` | Operations tab |
| `Ctrl+Shift+@` | Details tab    |
| `Ctrl+Shift+3` | Code tab       |

When focused on a text input, arrow keys move the text cursor normally. They navigate between elements only when the caret is at the boundary.
